STEP 2
Create a City Simulator.
The simulator must:
• generate passenger demand using Poisson distribution
• simulate taxi positions on a grid city
• track taxi movement
• track passenger pickups
City size should be configurable.
Example grid:
Z1 Z2 Z3
Z4 Z5 Z6
Z7 Z8 Z9
Each zone should store demand and taxi count.

STEP 3
Create a Gym RL Environment.
State should include:
• passenger demand map
• taxi positions
• time step
Action space:
• move north
• move south
• move east
• move west
• pick passenger
Reward design:
+10 passenger served
+5 high demand zone served
-1 idle movement
-10 passenger waiting too long
Environment must follow Gymnasium API.

STEP 4
Create a Reinforcement Learning Agent.
Use PPO from Stable-Baselines3.
Training script must:
• initialize TaxiEnv
• train PPO agent
• run 50k timesteps
• save trained model
Provide training visualization graphs.

STEP 5
Create FastAPI backend server.
Endpoints:
/train
Starts RL training.
/simulate
Runs taxi simulation and returns:
• taxi positions
• passenger demand map
• rewards
Response must be JSON.
